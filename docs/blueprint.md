# **App Name**: المسعف الذكي

## Core Features:

- User Authentication and Medical Profile: Users can register, log in, and manage their personal and medical information, including blood type, chronic diseases, allergies, and emergency contact details. This data is critical for emergency responses.
- Emergency Reporting System: A user-friendly interface with a prominent emergency button to quickly initiate distress calls, automatically sharing their precise location and pre-saved medical data. Quick action buttons allow specification of emergency type (e.g., accident, fire).
- Real-time Emergency Tracking: Once an emergency is reported, users can track the dispatched emergency vehicle's location on a map in real-time, along with the estimated arrival time and the designated receiving facility.
- AI-powered First Aid Advisor: An intelligent tool providing concise, step-by-step first aid instructions tailored to the reported or selected emergency type. It includes simple visuals and an option for audio playback.
- Emergency Dispatch Dashboard for Responders: A dashboard for emergency services (hospitals, civil defense) displaying active incident reports, caller location, and summarized medical data on an interactive map. Allows dispatchers to see nearest available units.
- Incident Management for Responders: Enables emergency responders to receive, review, accept, reject, or re-route emergency calls directly from the dashboard, updating the incident status in real-time.

## Style Guidelines:

- Primary action color: A vibrant emergency red (#FF0000) for critical actions like the 'Request Ambulance' button, conveying urgency and importance. This color anchors the primary interactions.
- Background color: Clean and pure white (#FFFFFF), enhancing readability and providing a sense of hygiene and professionalism.
- Success/Accent color: A bold green (#00B300) to signify successful operations, positive feedback, and status updates, providing a clear contrast to the primary red.
- Secondary color: A soft neutral gray (#808080) for backgrounds, subtle dividers, secondary text, and less prominent UI elements, ensuring clarity without distracting from core information.
- Headline font: 'Space Grotesk' (sans-serif), for a modern, tech-forward, and crisp aesthetic, suitable for clear titles and prominent text. Body font: 'Inter' (sans-serif), chosen for its exceptional readability across various screen sizes, ensuring critical information is easily digestible.
- Leverage a consistent set of clear, recognizable 3D medical icons to represent different conditions and features, aligning with a sophisticated and modern digital interface.
- A clean, modular layout utilizing 'Cards' for information display, providing distinct sections for medical profiles and first aid guidance. Incorporate subtle 'Glass UI' elements with soft, gentle shadows and predominantly circular call-to-action buttons for a contemporary and engaging feel.
- Introduce subtle and functional animations for user feedback, especially during critical actions like emergency report submission, status updates, and interactive elements, ensuring a responsive and reassuring user experience.
